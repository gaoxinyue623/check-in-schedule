# 📖 使用指南

## 欢迎使用"我的暑假计划表"！

这是一个专门为你设计的暑假学习和生活计划管理工具，让我们一起来学习如何使用吧！

## 🎯 主要功能

### 1️⃣ 查看每日计划

**步骤：**
1. 打开应用后，你会看到一周的日期（周一到周日）
2. 点击任意一天，就可以看到那天的所有任务
3. 每个任务都有时间和要做的事情

**小贴士：**
- 粉色高亮的就是你当前选中的日期
- 日期下方如果有星星，说明这天已经完成了一些任务

### 2️⃣ 完成任务得星星 ⭐

**步骤：**
1. 找到你刚刚完成的任务
2. 点击任务左边的小方框（复选框）
3. 看！方框里出现了勾勾 ✓
4. 右边的星星就是你获得的奖励！

**星星数量：**
- 学习任务：⭐⭐（2颗星）
- 阅读任务：⭐⭐（2颗星）
- 跳舞课：⭐⭐⭐（3颗星）
- 运动任务：⭐⭐（2颗星）或⭐（1颗星）
- 家庭劳动：⭐⭐（2颗星）

**重要提示：**
- 只有真正完成了任务才能打勾哦！
- 如果不小心点错了，再点一次就可以取消

### 3️⃣ 查看进度

在页面上方，你可以看到：
- **今日进度条**：显示今天完成了多少任务
- **今日星星数**：显示今天获得了多少颗星星
- **总星星数**：页面最上方的大大的星星数字

### 4️⃣ 兑换礼品 🎁

**步骤：**
1. 向下滚动页面，找到"礼品兑换"区域
2. 看看有哪些礼品：
   - 🎁 小玩具（10颗星）
   - 🍦 冰淇淋（15颗星）
   - 📚 新故事书（20颗星）
   - 🎬 看电影（25颗星）
   - 🎢 去游乐场（30颗星）
   - 👗 新衣服（40颗星）
3. 当你的星星够了，礼品卡片会变成黄色
4. 点击"兑换"按钮
5. 确认后，星星会减少，礼品就是你的啦！

**小贴士：**
- 可以先设定一个目标，比如"我要攒30颗星星去游乐场"
- 每次兑换都会记录次数，可以多次兑换同一个礼品

### 5️⃣ 添加新任务 ➕

如果你想添加新的任务：

**步骤：**
1. 点击"今日任务"旁边的"➕ 添加任务"按钮
2. 填写任务信息：
   - **时间**：比如 "14:00-15:00"
   - **活动内容**：比如 "画画"
   - **类别**：选择学习、阅读、运动、家务等
   - **奖励星星**：选择完成后能得几颗星（0-5颗）
3. 点击"保存"

### 6️⃣ 编辑任务 ✏️

如果想修改任务：

**步骤：**
1. 找到要修改的任务
2. 点击右边的铅笔图标 ✏️
3. 修改信息
4. 点击"保存"

### 7️⃣ 删除任务 🗑️

如果不需要某个任务了：

**步骤：**
1. 找到要删除的任务
2. 点击右边的垃圾桶图标 🗑️
3. 确认删除

### 8️⃣ 按类别查看任务

**步骤：**
1. 在任务列表上方，有几个彩色按钮
2. 点击不同的按钮，可以只看某一类任务：
   - 📋 全部
   - 📚 学习
   - 📖 阅读
   - ⚽ 运动
   - 🧹 家务
   - 🏠 日常

## 🌟 小技巧

1. **坚持打卡**：每天都要完成任务哦，这样星星会越来越多！

2. **设定目标**：选一个喜欢的礼品，算算需要多少星星，然后努力去完成！

3. **合理安排**：如果觉得某个时间的任务太难或太简单，可以和爸爸妈妈商量修改。

4. **诚实守信**：只有真正完成了任务才能打勾，这样才能培养好习惯！

5. **劳逸结合**：记得要有休息时间，不要一直学习哦。

## ❓ 常见问题

**Q: 我不小心点错了怎么办？**
A: 再点一次复选框就可以取消啦！

**Q: 我的星星不够兑换礼品怎么办？**
A: 继续努力完成任务，星星会越来越多的！

**Q: 可以一次兑换多个礼品吗？**
A: 可以的！只要星星够，想兑换几个都可以。

**Q: 如果我今天没有完成所有任务怎么办？**
A: 没关系，明天继续努力！重要的是养成好习惯。

**Q: 数据会丢失吗？**
A: 不会的！所有数据都保存在你的设备上，除非你清除浏览器数据。
