## 需要手动调整的template部分

由于文件较大，建议您手动进行以下调整：

### 1. 修改头部（第4-11行）
将：
```vue
<header class="header">
  <h1>🌈 寒假学习计划（1.24-2.20）🌈</h1>
  ...
</header>
```

改为：
```vue
<header class="header-fixed">
  <div class="header-content">
    <h1>🌈 寒假学习计划（1.24-2.20）</h1>
    <div class="star-counter pulse">
      <span class="star-icon">⭐</span>
      <span class="star-count">{{ totalStars }}</span>
      <span class="star-label">颗星星</span>
    </div>
  </div>
</header>
```

### 2. 添加主容器（第12行之后）
在header之后添加：
```vue
<div class="main-container">
```

并在文件末尾（Toast和ConfirmModal之前）关闭这个div

### 3. 修改日期导航（第14-29行）
将：
```vue
<div class="month-nav card">
```

改为：
```vue
<div class="month-nav-wrapper">
  <div class="month-nav">
```

并在对应位置添加关闭标签 `</div>`

### 4. 添加两栏布局
在进度卡片之前添加：
```vue
<div class="content-grid">
  <div class="left-column">
```

将任务列表放在left-column中

然后添加：
```vue
  </div>
  <div class="right-column">
```

将进度卡片和礼品卡片放在right-column中

最后关闭：
```vue
  </div>
</div>
```

### 5. 简化日期按钮内容
将day-btn内的内容简化为：
```vue
<div class="day-number">{{ day.day }}</div>
<div class="day-date">{{ day.date }}</div>
<div class="day-stars" v-if="getDayStars(index) > 0">
  ⭐{{ getDayStars(index) }}
</div>
```

### 6. 优化任务项结构
将task-item内的结构改为：
```vue
<div class="task-main">
  <input type="checkbox" .../>
  <div class="task-info">
    <div class="task-header-row">
      <span class="task-time">{{ task.time }}</span>
      <span v-if="task.stars" class="task-stars">
        {{ '⭐'.repeat(task.stars) }}
      </span>
    </div>
    <div class="task-activity">{{ task.activity }}</div>
    <div v-if="task.description" class="task-description">{{ task.description }}</div>
  </div>
  <div class="task-actions">
    <button class="btn-icon" @click="editTask(task)">✏️</button>
    <button class="btn-icon" @click="deleteTaskConfirm(task)">🗑️</button>
  </div>
</div>
```
