# 🌈 我的寒假计划表

一个专为**一年级小学生**设计的寒假作息学习计划表应用，通过可爱的界面、星星奖励机制和礼品兑换系统，帮助孩子养成良好的学习和生活习惯。

![Pinia](https://img.shields.io/badge/Pinia-2.1-FFD859?logo=pinia)

## ✨ 功能特性

### 📅 每日计划管理
- **一周视图**：清晰展示周一到周日的所有计划
- **详细任务**：每个任务包含时间、活动内容和详细说明
- **任务分类**：学习、阅读、运动、家务、日常等多种类别
- **完成打卡**：简单的复选框操作，完成即可打勾

### ⭐ 星星奖励系统
- **即时奖励**：完成任务立即获得星星
- **差异化奖励**：不同任务获得不同数量的星星（1-3颗）
- **进度可视化**：实时显示今日进度和累计星星数
- **每日统计**：周导航显示每天获得的星星数

### 🎁 礼品兑换
- **多样礼品**：小玩具、冰淇淋、故事书、看电影、游乐场、新衣服
- **星星兑换**：用累积的星星兑换心仪的礼品
- **兑换记录**：显示每个礼品的兑换次数
- **视觉反馈**：星星足够时礼品卡片高亮显示

### 🎨 友好界面
- **可爱配色**：温暖的粉色系主题，适合儿童
- **大字体**：清晰易读，适合一年级学生
- **动画效果**：平滑的过渡和悬浮效果
- **响应式设计**：支持手机、平板、电脑

### 💾 数据持久化
- **本地存储**：所有数据保存在浏览器本地
- **离线可用**：无需网络连接
- **自动保存**：每次操作自动保存

## 🚀 快速开始

### 环境要求
- Node.js 16+ 
- npm 或 yarn

### 安装依赖
```bash
npm install
```

### 开发运行
```bash
npm run dev
```
应用将在 `http://localhost:5173` 打开

### 生产构建
```bash
npm run build
```
构建产物在 `dist` 目录

### 预览构建
```bash
npm run preview
```

## 📱 打包成APP

本项目支持打包成iOS和Android应用，详见 [打包指南.md](./打包指南.md)

### 快速打包（使用Capacitor）

```bash
# 初始化Capacitor
npm run capacitor:init

# 添加iOS平台
npm run capacitor:add:ios

# 添加Android平台
npm run capacitor:add:android

# 在Xcode中打开iOS项目
npm run capacitor:open:ios

# 在Android Studio中打开Android项目
npm run capacitor:open:android
```

## 📖 使用说明

详细的使用说明请查看 [使用指南.md](./使用指南.md)

### 基本操作

1. **选择日期**：点击周导航中的日期切换不同的一天
2. **完成任务**：点击任务左侧的复选框标记完成
3. **查看进度**：查看进度条和星星数了解完成情况
4. **兑换礼品**：星星足够时点击礼品的"兑换"按钮
5. **添加任务**：点击"➕ 添加任务"按钮创建新任务
6. **编辑任务**：点击任务右侧的 ✏️ 图标编辑
7. **删除任务**：点击任务右侧的 🗑️ 图标删除

## 🛠️ 技术栈

- **框架**：Vue 3 (Composition API)
- **构建工具**：Vite 5
- **状态管理**：Pinia
- **路由**：Vue Router
- **样式**：原生CSS（CSS变量 + 现代特性）
- **数据存储**：LocalStorage

## 🎨 自定义主题

可以通过修改 `src/assets/main.css` 中的CSS变量来自定义主题颜色：

```css
:root {
  --primary-color: #FF6B9D;      /* 主色调 */
  --secondary-color: #FFA07A;    /* 辅助色 */
  --success-color: #98D8C8;      /* 成功色 */
  --warning-color: #FFD93D;      /* 警告色 */
  --bg-primary: #FFF5F7;         /* 主背景 */
  /* ... 更多变量 */
}
```

## 🔧 自定义功能

### 修改默认计划
编辑 `src/stores/schedule.js` 中的 `defaultDailySchedule` 数组

### 修改礼品列表
编辑 `src/stores/schedule.js` 中的 `defaultGifts` 数组

### 添加新的任务类别
在 `src/views/HomeView.vue` 中的 `categories` 数组添加新类别
